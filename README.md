- PHP 5.3+
- MySQL 5.5+
- libssh2 (`apt-get install libssh2 libssh2-1 libssh2-1-dev libssh2-php`)
- mind. 250 MB Speicherplatz
- PHP-Memory Limit:  mind. 64 MB.
