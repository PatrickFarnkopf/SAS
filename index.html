Was ist SAS?

Für viele stellt die Administration eines Linux-Servers eine Herausforderung dar. Dieses Problem haben wir erkannt und somit die Entwicklung eines einfachen, schnellen und innovativem Systems begonnen. Das Server Admin System ist eine Webanwendung, die auf einem separaten Server ausgeführt und im Webbrowser aufgerufen wird. SAS bietet Konfigurations- und Steuerungsmöglichkeiten für verschiedene Dienste und Systemeinheiten mit nützlichen Erklärungen für unerfahrene Benutzer. Jedoch ist das System Die Besonderheit von SAS besteht darin, dass SAS mehr als einen Server verwalten kann und der ferngesteuerte Server nur** minimale Sofwareanforderungen **hat.

Folgende Dienste sind nach der Installation zur Administration vorhanden:

  • Apache 2.2
  • MySQL-Server 5.5+
  • ProFTPD 1.3+
  • Samba 3.6+

Weitere Serversoftware kann mit ein grundlegendem Wissen in der Programmiersprache PHP (Version 5.4) realisiert werden, beispielsweise zur Steuerung eines Gameservers. Nach der ersten Stable-Version wird es auch Plugins für lighttpd und vsftpd geben.

Weitere Einstellungen betreffen das System selbst, wie etwa eine Rechteverwaltung, die anderen Benutzern nicht den vollen Funktionsumfang ermöglicht. Die Anwendung an sich erfordert einige Softwarepakete die zuvor über ein Skript einfach installiert werden. Anschließend leitet SAS den Benutzer durch eine simple Installationsroutine, mit der die Installation sehr einfach wird. Die Voraussetzungen an den Benutzer sind gering: Da nach den neusten Standards entwickelt wird, setzt SAS lediglich einen modernen Browser (beispielsweise die aktuellste Version von Google Chrome) und eine durchschnittliche Internetverbindung voraus. Nach der Anmeldung ruft der Benutzer die Serververwaltung auf. Nach der Serverauswahl wird eine Übersichtsseite aufgerufen die aktuelle Daten des Servers darstellt.

Auch die Vorraussetzungen an den zu verwaltenden Server sind durchaus gering. Wahlweise wird die Verbindung zum Server über das SSH2-Protokoll oder über einen sichereren und performanteren Daemon (selbstgeschrieben in “Ruby on Rails”) aufgebaut. Zusätzlich werden auch einige Pakete benötigt, die allerdings meistens schon vorinstalliert sind. Während der Entwicklung wurde klar, dass SAS aus Kompatibiltäts- und Zeitproblemen nur die Linux-Distribution "Ubuntu Server 12.04 LTS” und neuer, unabhängig von Architektur, vollständig unterstützt. Im Prinzip funktioniert jede andere Debian-basierte Distribution, jedoch wird nicht gewährleistet, dass diese vollständig unterstützt wird. Das gesamte System ist frei unter der Apache Lizenz v2.0 erhältlich und kann somit völlig kostenlos genutzt werden. Abschließend lässt sich sagen, dass das Server Admin System eine leistungsfähige Methode ist, seinen Server zu administrieren, egal ob Linux Anfänger oder Profi.

Systemanforderungen

Anforderungen für die Installation und Benutzung von SAS (Host):

  • PHP 5.4+
  • MySQL 5.5+
  • Wahlweise:
  • 1. libssh2 Extension (gibt es nur für 32-Bit Systeme) oder
  • 2. Ruby1.8
  • mind. 50 MB Speicherplatz
  • PHP-Memory Limit: mind. 64 MB

Verwendete Programmier- und Darstellungsprachen

PHP5 - Ruby - Bash - JS/jQuery - HTML5 - CSS3 - XML - SQL

Modulübersicht

Das Server Admin System verfügt über eine Vielzahl an Verwaltungsmöglichkeiten, die in verschiedene Module unterteilt sind.

Startseite

Die Startseite wird unmittelbar nach der Serverauswahl aufgerufen. Sie bietet verschiedene Möglichkeiten, um Systemdaten und Statistiken einzusehen.

Apache

Webserver-Verwaltung wird mit SAS einfach und schnell konfigurierbar. Die eingebauten Funktionen bieten die sichere Anpassung für individuelle Hostinglösungen und stellen genaue Statistiken dar, sowie eine einfache Option neue Kunden hinzuzufügen.

ProFTPD

Der FTP-Server wird mithilfe des Moduls effektiv administriert. Somit wird ProFTPD im Handumdrehen konfiguriert.

MySQL

Zur Verwaltung von MySQL Datenbank Systemen stellt das Server Admin System eine Reihe von Optionen zur Verfügung, mit denen der MySQL Server konfiguriert, seine Benutzer verwaltet sowie die Datenbanken administriert werden können.

Samba

Dieses Modul verfügt über das Potential einfach und effizient Freigaben zu erstellen und zu organisieren.

System

Der Managementbereich dient zur allgemeinen Verwaltung des Serversystems und beinhaltet Funktionen wie Neustart, Paket-Installation

Tools

Um Informationen über den Server herauszufinden gibt es hier verschiedene Module um Systeminformationen auszulesen.

Webuser

Hier werden verschiedene Möglichkeiten bereitgestellt, mit denen Zugriffsberechtigungen für einzelne Benutzer/Benutzergruppen gesetzt werden können.
