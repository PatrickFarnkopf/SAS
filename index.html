    Was ist SAS?

    SAS bedeutet „Server Admin System“ bzw. „Server Administration System“ und ist ein Webinterface zur Verwaltung von beliebig vielen Linux-Servern. Anders als bisherige Verwa ltungsoberflächen, muss diese nicht auf den betreffenden Server lokal installiert sein. SAS kann auf einen einfachen Webspace installiert werden. Das System kommuniziert über das sichere SSH2-Protokoll. Die Authentifizierung wird über einen RSA-Key oder einem speziellen Zugang vorgenommen. Über diese Verbindung werden die Befehle an den Linux Server gesendet und ausgeführt.

    Systemanforderungen

    Anforderungen für die Installation und Benutzung von SAS:

    PHP 5.3+
MySQL 5.5+
libssh2 Extension
mind. 250 MB Speicherplatz
PHP-Memory Limit:  mind. 64 MB.

    Hintergrund

    Mit SAS wird das Verwalten von einem sowie mehreren Linux Servern sehr einfach, besonders für Personen die sich mit der Verwaltung über die Konsole schwer tun. SAS ist einfach, übersichtlich und schnell. Auch für Profis steht die Möglichkeit der einfachen Erweiterung zur Verfügung, die es ermöglicht eigene Skripte einzubinden. Des weiteren ist SAS vollständig OpenSource und ist mit der MIT Lizenz frei erhältlich. Desweiteren ist SAS wie bereits beschrieben, einzigartig in seinem Wesen. Mehrere Linux Server werden über ein unabhängiges System ferngesteuert und somit sicher verwaltet.

    Modulübersicht

    Das Server Admin System verfügt über eine Vielzahl an Verwaltungsmöglichkeiten, die in verschiedene Module unterteilt sind.

    Startseite

    Die Startseite wird unmittelbar nach der Serverauswahl aufgerufen. Sie bietet verschiedene Möglichkeiten, um Systemdaten und Statistiken einzusehen.

    Apache

    Webserver-Verwaltung wird mit SAS einfach und schnell konfigurierbar. Die eingebauten Funktionen bieten die sichere Anpassung für individuelle Hostinglösungen und stellen genaue Statistiken dar, sowie eine einfache Option neue Kunden hinzuzufügen.

    Postfix

    Das Postfix Modul ermöglicht eine unkomplizierte Konfiguration eines Postfix E-Mail Servers. Es bringt Optionen zur Erstellung und Verwaltung von E-Mail Konten mit, sowie die Möglichkeit E-Mail Statistiken auszuwerten.

    FTP

    Der FTP-Server wird mithilfe des Moduls effektiv administriert. Somit wird ProFTPD im Handumdrehen konfiguriert.

    MySQL

    Zur Verwaltung von MySQL Datenbank Systemen stellt das Server Admin System eine Reihe von Optionen zur Verfügung, mit denen der MySQL Server konfiguriert, seine Benutzer verwaltet sowie die Datenbanken administriert werden können.

    Samba

    Dieses Modul verfügt über das Potential einfach und effizient Freigaben zu erstellen und zu organisieren.

    Management

    Der Managementbereich dient zur allgemeinen Verwaltung des Serversystems und beinhaltet Funktionen wie Neustart, Paket-Installation,

    Webuser

    Hier werden verschiedene Möglichkeiten bereitgestellt, mit denen Zugriffsberechtigungen für einzelne Benutzer/Benutzergruppen gesetzt werden können.

    Servertools

    Die SAS-Tools schaffen die Vorraussetzung für eine aufschlussreiche Darstellung der Serverinformationen und verdeutlicht die gegebenen Angaben, wie etwa eine grafische Darstellung der CPU-Auslastung.
